# Katniss Neural Processing

Pipeline de processamento e análise de curvas de luz astronômicas utilizando o algoritmo TLS
O projeto realiza:

- processamento de curvas de luz da missão Kepler
- limpeza e normalização de dados astronômicos
- extração de trânsitos utilizando TLS (Transit Least Squares)
- geração de visões globais e locais dos sinais

## Tecnologias e Algoritmos

- Python
- NumPy
- SciPy
- Pandas
- Lightkurve
- Transit Least Squares (TLS)
